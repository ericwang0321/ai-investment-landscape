1X Technologies

1X Technologies 是欧美人形机器人赛道里最明确押注“家庭场景”的公司。

1一句话结论

1X Technologies 是欧美人形机器人赛道里最明确押注“家庭场景”的公司。

它用 EVE 积累机构/工业经验，再用 NEO/NEO Gamma/NEO Home Robot 切入消费者家庭，并已经公开 $200 deposit、$20,000 购买价和 $499/月订阅口径。核心判断是，1X 的稀缺性在于“家用 humanoid + 软体安全设计 + Expert Mode 遥操作 + Redwood AI/1X World Model + Hayward 年产 10,000 台工厂”的组合；但它也是本组里最暴露于家庭隐私、安全、遥操作接受度和真实自主性不足的公司。投资上不能只看预订和视频演示，必须持续核验交付数量、退订/退款、Expert 人工成本、家庭任务成功率和数据授权。

1X 与 Figure、Agility、Apptronik 的差异很鲜明。Figure 和 Apptronik 先把 humanoid 带进汽车/物流/制造，Agility 则从仓储 tote handling 切入；1X 直接把 NEO 做成 household product。这个选择如果成功，TAM 和品牌心智极大，因为家庭机器人可以进入消费者日常；如果失败，风险也最大，因为家庭环境没有标准工位、没有安全围栏、隐私敏感、任务碎片化，而且早期自主性很可能不足。

1X 的产品语言也更像消费电子:柔软外观、针织外衣、低噪音、轻量、安全、陪伴、可对话、会自己充电、手机 App 管理。它不是在卖“更强的工业机械臂”，而是在卖“家里多一个可以干活的 humanoid”。这使它更接近 Tesla Optimus 的长期家庭叙事，但比 Tesla 更早公开消费者价格和订阅路径。

3业务模式与收费方式

3.1 NEO 购买 + 订阅

1X 官网截至 2026 年 7 月 6 日 显示 NEO Home Robot 可 Order，deposit 为 $200。公开媒体报道和维基整理口径显示，NEO 早期用户购买价为 $20,000，也计划提供 $499/month 订阅。这个价格结构非常关键，因为它给了家庭 humanoid 第一个可分析的单位经济锚。

按 $20,000 购买价看，NEO 不是大众家电，而是 early adopter / affluent household / research user 产品。按 $499/月订阅看，年化价格约 $5,988，如果含硬件、服务、维护、软件、远程 Expert 和保修，1X 需要控制硬件折旧、人工遥操作和现场服务成本。订阅模式理论上扩大可负担性，也让 1X 保留资产控制、持续更新能力和数据权限，但会让现金回收更慢。

3.2 Expert Mode / remote operation

NEO 官网明确写到，NEO 默认自主工作；对于它不会的复杂 chores，用户可以 schedule a 1X Expert 来远程指导，帮助 NEO 学习并完成任务。官网 FAQ 还把“1X Expert 是否能随时进入 NEO”“是否需要分享数据”“NEO 如何操作”等列为关键问题。New Yorker 2026 年文章也强调，NEO 仍大量依赖人类 tele-operators，用户需要接受有人通过 NEO 眼部摄像头进入家庭视角。

Expert Mode 是商业模式的双刃剑:

  • 正面:提升早期可用性，让 NEO 即使自主性不足也能完成任务；同时产生高质量 demonstration data。
  • 负面:人工成本可能吞噬订阅毛利；家庭隐私和远程访问会影响消费者接受度；若用户发现“机器人本质上是远程人类”，品牌信任会受损。

1X 的策略不是隐藏遥操作，而是把 Expert Mode 产品化、可预约、可提示状态。NEO 的 Emotive Ear Rings 用于显示状态，用户理论上知道何时由专家接入。真正需要核验的是用户是否愿意、数据是否可用于训练、专家人效如何、任务完成率是否提升到可规模化。

3.3 数据与软件升级

1X 的官网和 AI blog 反复强调 NEO “grows with you”。Redwood AI、1X World Model 和 World Model Lab 的商业意义是把家庭部署变成数据飞轮。早期用户不仅是客户，也是在不同家庭中为 NEO 提供真实长尾场景:厨房、衣物、宠物、儿童、家具、照明、杂物、玻璃、湿滑地面、声音和人的指令。

潜在收费方式包括:

  • NEO 本体购买价或月租；
  • Expert Mode 使用时长或包含在订阅中；
  • 软件能力升级包，例如 laundry、tidying、kitchen、eldercare；
  • 延保、维修、替换件、软体外衣和电池服务；
  • 企业/机构版 EVE 或 NEO deployment；
  • 数据授权和模型合作，但家庭数据必须有清晰 opt-in。

3.4 EVE / 企业场景

EVE 是 1X 早期 wheeled humanoid，面向 security、logistics、healthcare、industrial testing 等机构环境。EVE 的意义不只在收入，而在训练安全执行器、全身控制、远程操作流程和真实部署运维。相比 NEO 家庭业务，EVE/企业业务更可能采用客户合同、租赁或 RaaS，但公开材料没有给出标准价格和客户合同金额。

这些参数体现 1X 的设计哲学:NEO 不追求最大负载，而追求家庭安全、轻量、柔顺、低噪和人类空间适配。66 lbs 明显低于很多工业 humanoid，软体和 tendon-driven actuation 用于降低碰撞风险。IP68 hands、IP44 body、machine washable suit 等细节也更像家庭产品。

4.2 Redwood AI

Redwood AI 是 1X 面向 NEO 的 generalist AI model。官网称 Redwood 使 NEO 能移动、感知、交互并自主完成家庭任务。2025-06 的 Redwood mobility 更新称其 RL controller 支持自然行走、坐下、站立、跪下和爬楼梯，并可结合 stereo vision；同页还强调 Redwood 能训练成功和失败样本、处理 never-before-seen objects、结合 off-board language model 理解用户意图。

Redwood 的核心挑战是家庭任务长尾。工业机器人可以固定“搬一个 tote 到 conveyor”，家庭机器人要处理“把这个地方收拾一下”“把衣服叠好”“把杯子放回柜子”“厨房台面擦干净”这类目标模糊、环境变化大的任务。1X 因此把世界模型作为 AI 路线重点。

4.3 1X World Model

1X 2025-06 和 2026-01 的 World Model 文章解释了其技术路径。1XWM 是 video model grounded in physics，用于预测 NEO 动作后世界如何变化。2026-01 文章更进一步，把 world model 集成到 NEO policy:

  • 使用 text-conditioned video generation 生成未来视频；
  • World Model backbone 建在 14B generative video model 上；
  • 先用 900 小时 egocentric human video 做 mid-training；
  • 再用 70 小时 robot data 做 embodiment fine-tuning；
  • IDM 用 400 小时 unfiltered robot data 训练，从生成帧反推出动作；
  • 文中展示抓取、双手、人机互动、新任务泛化等实验；
  • 部分任务依然存在深度理解、物体一致性和成功率不足问题，例如 pull tissue 任务多次 rollouts 后成功率从 30% 提高到 45%。

这份技术披露很有价值，因为它没有只讲“机器人很聪明”，也承认生成 rollout 可能视觉上合理但物理上失败。1X 的路线是用互联网视频 + egocentric human video + NEO 真实数据降低对大规模 teleoperation 的依赖。如果成立，家庭数据收集成本会下降；如果不成立，NEO 仍要大量依赖 Expert/遥操作，单位经济性会很差。

4.4 NEO Factory / 制造

1X 官网称 Hayward NEO Factory 有年产 10,000 台 NEO 的 capacity。2026 年文章和媒体报道提到 58,000 平方英尺级别设施、约 200 名员工、NEO 机器人参与轻量搬运/分拣/取料等生产辅助。与 Figure BotQ 一样，“机器人建造机器人”是叙事重点，但当前更现实的理解是人类工人仍承担主要组装、测试和质量控制，NEO 辅助部分物料流。

NEO 工厂能力的关键核验点是实际出货。年产能是设计能力，不代表 2026 会交付 10,000 台。家庭机器人还需要包装、安装、远程支持、维修、更换软体、电池服务和客户教育，这些都比工厂内 RaaS 更复杂。

1X 的商业化判断要同时看“订单/预订”和“可用性”。家庭用户愿意支付 $200 deposit 不等于愿意承担 $20,000 购买、长期订阅、家庭数据分享和 Expert 远程进入。真正强的指标应该是已交付 NEO 数量、活跃家庭数、每周完成任务数、每任务人工介入比例、订阅留存、退款率、事故率和 NPS。
