Kling AI / 可灵AI

Kling AI/可灵 AI 是快手体系内孵化出的 AI 视频生成业务线，已从 2024 年快影内测工具升级为全球化 web/app/API 产品，并在 2026-07 通过快手推动的外部融资和潜在拆分上市进入独立资本化阶段。

1一句话结论

Kling AI/可灵 AI 是快手体系内孵化出的 AI 视频生成业务线，已从 2024 年快影内测工具升级为全球化 web/app/API 产品，并在 2026-07 通过快手推动的外部融资和潜在拆分上市进入独立资本化阶段。

与 Runway、Luma、Pika、Midjourney 不同，可灵早期不是普通 VC 创业公司，不能把快手内部投入写成传统融资轮；截至 2026 年 7 月 6 日，最重要的资本事件是 WSJ/快手披露线索称可灵获得约 RMB19.04B 外部注资，估值约 $18B，快手持股可能降至最低约 68.33%。投资判断是，可灵的优势是快手短视频/快影/广告生态、低价高频迭代、Kling 3.0 级别视频模型和中国供应链/算力组织能力；风险是独立财务口径尚少、商业化仍早、内容审核/地缘合规限制、与字节 Seedance/即梦、OpenAI Sora、Google Veo、Runway、Luma 的全球竞争，以及拆分上市能否把大厂内部业务真实转化为独立公司治理和收入。

可灵是本组里最需要谨慎写资本化口径的公司。Runway、Pika、Luma 是独立创业公司;Midjourney 是自举式独立公司;可灵则是快手的战略业务。2024–2025 年的研发、流量、用户、模型训练、产品入口和商业资源都来自快手体系，因此不能把“快手内部投入”当作“可灵融资”。只有当快手将可灵业务引入外部投资、降低持股、准备拆分/上市时，才形成可灵层面的资本化事件。

产品层面，可灵的定位已从“中国版 Sora 竞品”变成全球低成本视频生成平台。它与快手短视频、快影剪辑、广告主、短剧生产、直播电商和中国内容生态天然相关，这给它带来 Runway/Luma/Pika 难以复制的场景。但全球化时，可灵也面临中国监管、内容审查、海外信任和地缘合规问题。

4业务模式与收费方式

4.1 Creator & Team / Membership credits

Kling AI 国际站具有 membership、video generator、image generator、lip sync、effects template、community 等入口，商业模式包含会员和 credits。由于官方 pricing 页面为动态渲染，公开抓取时无法稳定得到完整套餐价格表;研究中不把第三方转载的套餐价格写成确定官方报价。可回源的结论是:

  • 用户通过 membership / credits 消耗生成视频和图片。
  • 视频生成按模型、清晰度、时长、模式、是否加速等消耗 credits。
  • Kling 以“低价、更新快”获得用户和媒体关注。
  • 个人创作者和小团队是重要用户层。

与 Pika 相比，可灵更强调模型能力和低价;与 Runway/Luma 相比，Kling 更像“高性价比视频生成平台 + 大厂生态产品”;与 Midjourney 相比，它从视频而不是图片审美起家。

4.2 API access

Kling 官网业务表单清楚把 API Access 列为用户类型之一，并要求用户说明是否:

  • Not tested yet
  • Tested only on the website
  • Tested via API call
  • Already integrated in production

这说明可灵已把 API 作为商业化路径，而不只是 consumer web app。表单还要求选择 Capability Requirements: Video Generation、Image Generation，并填写 use case。示例场景包括 creative platform 使用 Kling API 做 text-to-video、image-to-video、character animation，帮助用户更高效创建 short-form content。

4.3 企业/API 预算档

官网 business inquiry 表单给出 API 预算/使用阶段:

API usage stage月预算档商业含义
Initial exploration or personal testingtypically under $2,000/month个人/小规模试用
Early-stage testing or pilot deploymenttypically $2,000–$5,000/month试点项目
Growing product or team usagetypically $5,000–$30,000/month成长期产品/团队

这张表非常重要。它说明可灵的 API 商业化不是只卖几美元 credits，而是明确在筛选 $5k、$30k、$150k+/month 的 B2B 预算。相比 Pika 通过 Fal.ai 分发 API、Runway/Luma 直接列 API pricing，可灵的方式更像“表单收集需求 + 商务报价 + 大客户折扣”。

4.4 Enterprise plan / partnerships

官网业务表单的 user type 包括:

  • Creator & Team Plan: individual creators and small/medium studios that need collaborative workflows。
  • Enterprise Plan: bulk purchasing, asset management, no-code deployment。
  • API Access: developers and businesses seeking flexible access to core generation capabilities and solution APIs。
  • Partnerships: model distribution and market expansion。

这说明可灵的企业化方向包括:

  • 批量购买 credits。
  • 团队/资产管理。
  • 无代码部署。
  • API 接入。
  • 模型分发和渠道合作。

对快手而言，企业/合作客户可能包括广告代理、短剧公司、MCN、内容平台、AI 工具聚合商、智能硬件、游戏和电商。可灵不一定需要像 Runway 那样先从 Hollywood studio 获取标志性客户，因为快手内部本身就有短视频/广告/直播电商场景。

4.5 快手生态内商业化

可灵的一个独特收入来源是快手内部生态:

  • 快影/剪辑工具可把 AI 视频能力分发给创作者。
  • 快手广告主可用 AI 生成素材、短视频广告、直播电商素材。
  • 短剧和内容生产者可用 AI 生成分镜、预告、封面、虚拟场景。
  • 快手平台可用 AI 工具提升创作者留存和广告转化。

问题在于:内部生态贡献到底是独立收入、集团内部结算、用户留存工具，还是战略投入? 公开资料没有给出拆分后的 revenue recognition 规则，因此不能直接把快手生态价值全部计入可灵收入。

5产品与技术

5.1 Kling Video

Kling Video 是核心产品，覆盖 text-to-video、image-to-video、multi-image video 等。早期公开资料称可灵可生成较长视频、30fps、1080p 等能力，并因与 OpenAI Sora 对比而出圈。后续 Kling 2.0/2.1/3.0 持续迭代，强调运动、物理、镜头、清晰度、多主体和故事能力。

公开资料对 Kling 3.0 的描述包括:

  • up to 4K。
  • 15 seconds。
  • 30–60fps。
  • Video 3.0、Video 3.0 Omni、Image 3.0、Image 3.0 Omni。
  • multi-shot storyboarding, up to 6 cuts per generation。
  • native audio in English, Chinese, Japanese, Korean, Spanish。
  • character voice and appearance cloning via reference video。

这些能力如果稳定，会让可灵从短 clip generator 向 storyboard / ad / short drama / social content production 迈进。尤其是 multi-shot 和 native audio，直接瞄准 Runway、Luma、Veo 和 Sora 的高端能力。

5.2 Kling Image

Kling Image 使可灵不只是视频工具，而是图像+视频一体平台。图像生成可用于:

  • 视频首帧。
  • 角色/产品 reference。
  • 广告 key visual。
  • 电商商品图。
  • 短剧海报/封面。
  • 生成后转视频。

这与 Luma Uni+Ray、Runway Gen-4 Image+Video、BFL Flux+视频工具的工作流一致。视频生成平台都需要强图像能力，因为高质量 starting frame 和 reference 是视频可控性的基础。

5.3 Lip Sync / Effects Template

Kling 的产品菜单包括 Lip Sync 和 Effects Template。Lip Sync 对短视频、短剧、虚拟角色、广告口播和跨语言本地化有直接价值;Effects Template 则服务更轻量的社媒创作者。这个组合使可灵覆盖从专业生产到娱乐效果的两端:

  • 专业端:广告、短剧、影视动画、口播本地化。
  • 消费端:模板、特效、社区传播。

5.4 技术路线

公开资料称 Kling 使用 diffusion transformer 架构，并有自研 3D VAE 网络用于时空压缩。这类架构方向与全球视频模型趋势一致:DiT、latent video representation、spatiotemporal compression、motion modeling、multi-modal conditioning。可灵背靠快手的优势在于:

  • 大规模视频数据和内容理解经验。
  • 短视频推荐、剪辑、特效和创作者生态。
  • 工程化推理和内容安全能力。
  • 中国本地产品迭代速度。

技术风险也相应存在:训练数据来源、版权、跨境数据、内容审核和模型透明度都会影响海外企业客户。

5.5 与快影/快手的结合

可灵早期通过快影等产品触达用户，这一点和 Runway/Pika/Luma 从独立 web app 起步不同。快影是视频剪辑工具，天然处在创作者工作流中;快手是短视频平台，天然有内容分发和广告转化。可灵如果嵌入这些场景，可以形成“生成 -> 剪辑 -> 发布 -> 分发 -> 广告”的闭环。

但拆分上市后也会有问题:可灵与快手之间如何定价? 数据、流量、用户和成本如何结算? 这些会影响独立财务报表。
