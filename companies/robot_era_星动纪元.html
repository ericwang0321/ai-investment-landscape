机器人/具身智能 应用 中国

Robot Era / 星动纪元

星动纪元是中国具身智能/人形机器人里最强调“清华背景 + 软硬件全栈 + 高端客户交付”的公司之一。

1一句话结论

星动纪元是中国具身智能/人形机器人里最强调“清华背景 + 软硬件全栈 + 高端客户交付”的公司之一。

官网 SEO 与前端包口径显示，公司为清华大学持股的具身智能企业，专注软硬件全栈自研，已完成近 5 亿元 A 轮融资、实现产品规模化交付，并称全球市值 TOP10 科技公司中 9 家为其客户；产品线包括 L7 全尺寸双足人形、M7 全尺寸半身人形、Q5 轮式服务机器人、XHAND 1 / Lite / Pro 灵巧手。投资判断是，星动的潜在价值在于把大客户真实场景、VLA 模型、全尺寸本体、半身操作平台和灵巧手供应链结合起来；最大疑点是客户口径、订单金额、收入确认、毛利率、量产台数和融资估值细节仍需要用融资材料、客户访谈或未来招股书核验。

4业务模式与收费方式

4.1 整机销售:全尺寸双足、半身操作平台、轮式服务机器人

星动的产品组合覆盖三类本体:

产品定位可能收费方式商业意义
L7新一代全尺寸双足人形机器人整机销售、项目制交付、工业/科研定制、服务合同品牌和全栈能力旗舰，适合展示双足运动和通用任务
M7新一代全尺寸半身人形机器人半身操作平台销售、数据采集/操作场景交付去掉双足复杂性，聚焦上肢操作和桌面/工站任务
Q5轮式人形/轮式服务机器人整机销售、商业服务、教育/公共服务项目更易部署、更安全，适合早期商业化
XHAND 系列灵巧手单独零部件销售、随整机配置、供应链合作可进入其他机器人公司和科研市场

这个组合比只做人形本体更稳。双足机器人最难量产和安全部署，半身/轮式平台更容易进入客户真实场景。M7 和 Q5 可以帮助公司积累操作数据和客户反馈，再迁移到 L7。

4.2 企业项目制:大客户场景验证和定制交付

官网称全球市值 TOP10 科技公司中 9 家为客户。即使这只是“合作/试用/采购/PoC”中的宽口径，它仍说明星动在头部客户触达上有强能力。大客户项目的收费可能包括:

  • 样机采购；
  • PoC / 场景适配费；
  • 数据采集服务；
  • 机器人本体和灵巧手定制；
  • 现场部署、运维和培训；
  • 软件/模型适配与升级；
  • 长期维保。

但大客户项目也有财务风险:验收周期长、定制多、议价强、收入确认慢。星动需要把项目能力沉淀成可复用产品，而不是每个客户都重做一套。

4.3 灵巧手和部件供应

XHAND 1 / Lite / Pro 是星动最可能形成独立供应链收入的产品线。人形机器人真正进入工业和家庭，关键不是走路，而是能不能稳定抓取、拧、插、拉、按、夹、递送、整理。灵巧手是机器人本体公司的共性需求，也可以被科研机构和其他本体厂采购。

收费方式可能包括:

  • 单独销售 XHAND；
  • 作为 L7/M7/Q5 的高配选项；
  • 向合作伙伴提供 SDK、驱动和控制算法；
  • 与客户共同开发特定夹爪/传感器配置。

4.4 软件、模型、数据与 RaaS

前端包中出现 VLA / embodied model / data collection 等表述，说明星动把模型和数据作为产品逻辑的一部分。未来收入可能包括:

  • VLA 模型授权；
  • 按场景任务包收费；
  • 数据采集和标注服务；
  • 机器人管理平台订阅；
  • RaaS 按月/按任务/按工位收费。

目前公开资料没有给出价格表，因此财务模型只能做敏感性框架，不能编造 ARR 或单机毛利。

5产品与技术

5.1 L7:全尺寸双足人形旗舰

前端包口径显示 L7 为“新一代全尺寸双足人形机器人 / Full-size Bipedal Humanoid Robot”。其产品叙事包括 factory screwing、data collection、platform validation、VLA 大模型、全身/上身配置切换等。L7 的商业意义在于证明星动具备完整机器人系统能力:

  • 下肢双足运动控制；
  • 上肢灵巧操作；
  • 感知、规划、控制闭环；
  • 数据采集与模型训练；
  • 工业场景任务验证。

L7 对投资估值的贡献主要来自“上限”:如果全尺寸双足能在真实工厂或大客户场景连续工作，公司可被重估为通用人形机器人平台。但 L7 也最难商业化，安全、成本、跌倒、续航和维护都比轮式/半身平台更复杂。

5.2 M7:全尺寸半身人形

M7 前端包口径为“Full-Size Half-Body Humanoid Robot / 新一代全尺寸半身人形机器人”。它保留双臂、躯干、灵巧手和感知/控制，但减少双足行走的复杂度。对真实商业化来说，M7 可能比 L7 更早落地，因为许多高价值任务发生在固定工位:

  • 桌面分拣；
  • 扫码；
  • 插拔/拧紧；
  • 质检；
  • 实验室操作；
  • 数据采集；
  • 机械臂替代或补充。

半身平台的优势是稳定、安全、续航和控制成本更低；劣势是移动能力弱，需要与移动底座或工作站结合。它适合先做“手能干活”，再逐步扩展到“人形能移动”。

5.3 Q5:轮式服务机器人

前端包显示 Q5 是 Wheeled Humanoid Robot，并出现 education scene、business service 等场景。Q5 很可能承担早期商业服务和教育场景落地:

  • 科技馆/展厅；
  • 教育演示；
  • 商业服务；
  • 导览接待；
  • 数据采集和平台验证。

与 L7 相比，轮式平台在公共空间更安全、更可控，也更容易过客户验收。它可能不是估值上限最高的产品，但可能是收入质量更早可验证的产品。

5.4 XHAND:灵巧手产品线

前端包显示 XHAND 1、XHAND 1 Lite、XHAND 1 Pro，并出现“1100g”“空载抓握寿命 100 万次”等字段。灵巧手的关键指标通常包括:

  • 自由度；
  • 单指/整手负载；
  • 重量；
  • 响应速度；
  • 力控/触觉传感；
  • 开合寿命；
  • SDK 和控制接口；
  • 与不同机械臂/人形本体兼容性。

XHAND 若能独立销售，星动可以不只依赖整机。尤其在中国灵巧手供应链快速发展的 2025–2026 年，部件公司和本体公司之间的边界会变得模糊:本体公司自研灵巧手可以降本，也可以外供。

5.5 VLA、数据和全栈自研

官网标题称“行业唯一软硬件全栈自研具身智能”，SEO 称公司致力于打造通用智能体。前端包中出现 VLA 大模型、data collection、platform validation 等产品语言。星动的技术路线可以理解为:

  1. 用 L7/M7/Q5/XHAND 在客户场景中采集真实数据；
  2. 用 VLA / embodied model 学习任务；
  3. 在半身或轮式平台先稳定部署；
  4. 再迁移到全尺寸 L7；
  5. 通过大客户场景扩大任务覆盖。

这条路线与 X Square、智元类似，但星动强调清华和高端客户，而 X Square 强调开源模型/数据，智元强调量产和 AgiBot World。

商业化判断:星动公开信息呈现强势，但“客户数”不能直接等于“收入质量”。大客户往往会做多家机器人供应商并行测试，采购样机也可能金额不大。需要把客户关系分成接触、试用、PoC、正式订单、规模部署、复购六层核验。

对星动尤其要重视“客户验收路径”。如果 TOP 科技客户购买的是 L7/M7/Q5/XHAND 样机，早期收入确认可能来自一次性硬件；如果客户把机器人放入真实工站、实验室或展厅长期运行，则会产生维保、动作/模型迭代、数据采集和二次采购。两者在财务模型中完全不同。前者证明品牌触达，后者证明产品市场契合。星动后续最有价值的公开进展，不是又发布一个跳舞或互动视频，而是披露某个客户场景的连续运行小时、任务成功率、人工介入比例和复购台数。

7单位经济性 / 关键财务逻辑

星动的单位经济性可以拆为四个产品层级。

L7 全尺寸双足:单机 ASP 可能最高，但 BOM、调试、售后和安全成本也最高。适合作为旗舰和大客户验证，不一定是最早盈利产品。

M7 半身平台:去掉双足后，单位成本和安全风险下降，任务集中在上肢操作。若能进入固定工站，部署经济性可能优于 L7。

Q5 轮式服务:移动稳定性和公共场景接受度更好，适合商业服务和教育，但价格上限和任务复杂度低于工业操作。

XHAND:部件业务有潜在高周转和较好毛利，但需要量产一致性、寿命和兼容性。若能独立外供，收入弹性更强。

核心财务问题:

  • 大客户采购的是样机、PoC 还是规模订单；
  • L7/M7/Q5/XHAND 各自 ASP；
  • 每个客户部署需要多少工程师天数；
  • 灵巧手是自用为主还是外供为主；
  • 交付后的复购率；
  • 软件/模型/数据收入能否独立计费；
  • 规模化交付是否带来毛利改善。

如果把星动拆成“旗舰展示 + 工站操作 + 轮式服务 + 灵巧手外供”四个收入池，最先验证的可能不是 L7，而是 M7/Q5/XHAND。L7 的价值在于证明公司能做完整人形系统，并在客户面前形成品牌溢价；M7 更适合固定工站任务，可能更早形成可验收 ROI；Q5 适合教育和商业服务，部署风险更低；XHAND 可独立进入其他本体和科研平台。理想状态下，公司用 L7 获客和展示技术，用 M7/Q5 形成早期收入，用 XHAND 扩大供应链触点，再用客户数据反哺 VLA 模型。若四条线互相割裂，研发和交付复杂度会快速上升。

8竞争格局

8.1 与智元

智元优势在 A2/G/D/C5/OmniHand 多产品矩阵、量产工厂、AgiBot World 和 1,000 台/15,000 台量产线索。星动优势在清华背景、高端客户口径和全栈自研叙事。两者都在抢工业客户、数据、灵巧手和人形落地样板。

8.2 与宇树

宇树以价格和出货心智压制行业，G1/R1 价格极低。星动不适合与宇树打最低价，而应强调高端客户、任务能力、灵巧操作和定制交付。若客户愿意为稳定任务付费，星动可以避开价格战；若客户只是买开发平台，宇树更有优势。

8.3 与 X Square

X Square 和星动都强调模型、本体、灵巧手和数据。X Square 的 WALL-A、Quanta X1/X2、XRZero-G0 更偏开源模型/数据平台；星动的 L7/M7/Q5/XHAND 更偏全栈产品和客户交付。两者会在机器人大脑和灵巧操作上直接竞争。

8.4 与破壳 / 松延动力

破壳走消费级普惠路线，Bumi / N2 / E1 / 小诺 / W1 产品覆盖家庭、教育、仿生人形，官网新闻称 B 轮累计近 10 亿元。星动更偏高端客户和工业/科研，二者短期客户不同，但在教育和商业服务会交叉。

8.5 与 Tesla Optimus、Figure、1X、优必选、傅利叶

Tesla 拥有工厂数据、执行器、电池和量产能力；Figure 有 BMW、Helix 和资本；1X 聚焦家庭；优必选有上市公司和 Walker S 工业客户；傅利叶有康复场景积累。星动需要用客户交付和任务能力证明自己不只是中国版跟随者，而是有本土高端客户和灵巧操作优势。

9团队与背景

公开可抓取资料显示星动纪元强调清华大学持股。清华背景的价值主要体现在:

  • 机器人、控制、AI 和工程人才池；
  • 科研资源和校友网络；
  • 对大客户和政府/产业资源的可信度；
  • 技术叙事和融资背书。

但后续还需要核验:

  • 创始人、CEO、CTO、硬件、模型、供应链和销售负责人的具体履历；
  • 团队规模和研发占比；
  • 是否有大规模硬件量产经验；
  • 是否有工业客户交付负责人；
  • 是否有安全和质量体系负责人。

机器人公司是系统工程，单一学术背景不能替代制造、供应链和售后能力。

10主要风险

客户口径风险:TOP10 科技公司 9 家为客户是很强表述，但需要确认是付费客户、试用客户、合作伙伴还是接触客户。不同口径对估值意义完全不同。

交付风险:规模化交付需要明确台数、验收、运行小时和复购。机器人样机交付不等于生产级部署。

技术风险:L7 双足、M7 半身、Q5 轮式和 XHAND 都需要不同控制栈。全栈自研范围越大，研发压力越高。

成本风险:高端产品如果大量定制，短期毛利可能被工程服务吞噬。灵巧手寿命、传感器和驱动成本也会影响毛利。
